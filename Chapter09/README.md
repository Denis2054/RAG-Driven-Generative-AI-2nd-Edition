**Copyright 2024, 2025, 2026, Denis Rothman**

## Overview

This chapter demonstrates building a Conversational RAG (Retrieval Augmented Generation) Video Store Agent. The video dataset was generated by Denis Rothman with the OpenAI Sora diffusion transformer model on inVideo specifically for a Video Production educational use case.

The chapter includes:
- Video dataset visualization
- Processing and analyzing AI-generated sports and activity videos
- Frame extraction from videos for analysis
- CSV-based comment/metadata files for each video
- Interactive notebook for exploring the video dataset

## Content Categories

### Videos Dataset
The videos directory contains AI-generated videos across multiple sports and activities:
- **Basketball**: 5 videos (basketball1-5.mp4)
- **Football/Soccer**: 5 videos (football1-2.mp4, soccer_pass.mp4, soccer_player_head.mp4, soccer_player_running.mp4)
- **Other Sports**: Hockey, Skiing, Swimming, Surfing
- **Activities**: Jogging, Walking, Alpinist/Climbing
- **Special**: Introduction video with audio (AI_Professor_Introduces_New_Course.mp4)

### Comments Directory
Contains CSV files with metadata and annotations for each video file. These files provide structured data about each video for RAG applications.

### Frames Directory
Contains extracted frames from video files for frame-by-frame analysis. Currently includes 222 frames from basketball3.mp4, stored as individual JPEG images.
